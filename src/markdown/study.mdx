## 개발 전까지 과제

테스트 계정 - this / that

### 프론트

아래 기능들이 모두 가능하여야 함

- 로그인
    - username와 password를 이용한 로그인
    - 앱을 재시작해도 로그인 유지
- 홈
    - 할일 목록(각 row는 할일 제목, 체크박스, 삭제버튼이 포함되어야 함)
    - FAB(Floating Action Button)을 통해 할일을 추가하는 dialog를 보여줄 수 있어야 함
    - Appbar 오른쪽에 로그아웃 버튼을 통해 로그아웃할 수 있어야 함

### 백엔드

제공된 API 문서의 아래 기능들이 가능하여야 함

**볼드 처리된 route들은 auth가 있어야 접근 가능해야 함**

    - /auth
        - /login - post
        - /register - post
    - **/user**
        - **/profile - get**
    - **/todo**
        - **- get**
        - **- post**
        - **- patch**
        - **- delete**
